# esper
A very minimal pseudo random number generator with union's data packing. <br>

It's available both in form of both x86 and x64 variants. <br>

# Requirement(s)
1. C/C++ compiler, C99/C++11. <br>
2. CMD/Terminal (if needed). <br>
